Introduction to SnowGem

SnowGem (XSG) is a fork of ZCash (ZEC). It has a fast block time, cheap transactions, active dev team and community, and is the only equihash coin with masternodes.

SnowGem's technology

SnowGem is an equihash coin, so it has the ASIC (Application Specific Integrated Circuit) resistance of coins like BitcoinZ and ZelCash. This means network hash is unlikely to be centralized with a few high-end ASIC miners. Anyone with a GPU or CPU can mine and receive XSG.

It also uses zk-SNARKS, which allows transactions to be stored on the blockchain completely encrypted, while still allowsing for verification and validation. In simpler terms, transactions can be proved valid without revealing any information about them. This allows for anonymous transactions along side standard transparent transactions.

SnowGem's latest developments

Masternodes went live in April 2018 and started paying out on the SnowGem network. This makes SnowGem unique as it's the first equihash coin to implement masternodes, of which there are now over 150 across the globe.

What are masternodes?

The function of masternodes varies between implementations. In some cases they perform special functions on the network that aren't handle by regular nodes. At the very least they help strengthen the network because they're (theoretically) always available, sharing their peers with the network, and they're constantly maintained. Because they play an important role, running a masternode requires stake, but in return the owner is paid a portion of the block reward and/or fees.

Masternodes provide an alternative way to earn coins without having to mine them. Instead of buying mining equipment, you set up a masternode on small server, stake your coins (10,000 XSG) and watch the funds roll in. That's the simple version. Learn more about setting up SnowGem masternodes on their website.

Something to note about the implementation of masternodes in SnowGem: it reduces the payout to miners. While the block reward may be 20 XSG, 35% of that (7 XSG) goes to the owner of the maternode. The percentage taken by the masternode increases by 10% every 43,200 blocks until it reaches a 50/50 split between masternode and miner(s). Of course, if the price continues to increase (and it looks like it will), mining SnowGem still looks like a worthwhile investment.

Thoughts on SnowGem

SnowGem's implementation of masternodes is interesting, if not somewhat controversial. Masternodes can leave those who are poorer at a disadvantage, because of the coins required upfront to activate them. But with the value of SnowGem increasing, even with the cut taken by masternodes, if the difficulty and network hash is right, mining can still be very profitable.
